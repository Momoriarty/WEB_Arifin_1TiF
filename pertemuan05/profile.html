Deskripsi Mata Kuliah

Tentang Mata Kuliah

Bengkel Pemrograman Web adalah mata kuliah yang dirancang untuk memberikan mahasiswa keterampilan praktis dalam pengembangan aplikasi web modern. Mata kuliah ini menggabungkan teori dan praktik untuk memastikan mahasiswa siap menghadapi tantangan dunia nyata dalam pengembangan web.

Durasi: 16 minggu
SKS: 3 (1 Teori, 2 Praktikum)
Prasyarat: Pemrograman Dasar
Materi Pembelajaran
  • Dasar-dasar HTML5, CSS3, dan JavaScript modern
  • Pengembangan aplikasi dengan React dan Next.js
  • Styling dengan Tailwind CSS dan komponen UI modern
  • Integrasi API dan manajemen state
  • Deployment dan hosting aplikasi web
